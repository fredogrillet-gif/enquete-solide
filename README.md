# L'Affaire du Musée des Formes 🔍

**Jeu d'enquête géométrique — CM1 / CM2**

Jeu pédagogique sur les solides (polyèdres, formule d'Euler, patrons…).

## 🚀 Déploiement

### GitHub Pages
1. Uploader tous les fichiers à la racine du dépôt
2. Activer GitHub Pages depuis `Settings > Pages > Branch: main / root`
3. L'URL sera : `https://[utilisateur].github.io/[depot]/`

### Google Sites
1. Dans Google Sites : **Insérer > Intégrer**
2. Coller l'URL GitHub Pages
3. Ajuster la taille de l'iframe (recommandé : largeur 100%, hauteur 900px)

## 📁 Structure
```
index.html   ← fichier principal (SPA tout-en-un)
404.html     ← identique à index.html (fallback GitHub Pages)
.nojekyll    ← désactive Jekyll sur GitHub Pages
README.md    ← ce fichier
```

## ✅ Compatibilité
- ✅ GitHub Pages
- ✅ Google Sites (iframe)
- ✅ Aucune dépendance externe sauf Google Fonts
- ✅ Fonctionne sans serveur (fichier local aussi)
